# Employee-Database

Calculates the salary/wage of three kinds of Employees:
  Commissioned Employee (Fixed Rate per month)
  Hourly Employee (Rate is per hour)
  Piece Worker (Rate is per piece sold)
  
No API included. Written in pure Java.
